# Hărți de zgomot pentru aeroporturi principale din România

Acest proiect oferă o vizualizare web a hărților strategice de zgomot pentru aeroporturile principale din România, realizate conform Directivei 2002/49/CE privind evaluarea și gestionarea zgomotului ambiental.

## Vizualizare hartă

👉 https://zgomot.github.io/zgomot-aeroport/

## Tip serviciu

- Serviciu: WMS (Web Map Service)
- Standard: OGC WMS 1.1.1
- Sursă: GeoNoise MMAP

## Descriere

Sunt disponibile următoarele layere:

- **Lden** – nivel de zgomot pentru zi-seară-noapte  
- **Lnight** – nivel de zgomot pentru perioada de noapte  

Aceste layere reprezintă contururi de zgomot (noise contours) calculate pentru aeroporturile principale, inclusiv zonele din aglomerările urbane afectate.

## Instituție responsabilă

Ministerul Mediului, Apelor și Pădurilor (MMAP)

## Observații

Aceasta este o vizualizare web pentru explorarea datelor.

Pentru utilizare oficială și descărcarea datelor, utilizați serviciile INSPIRE (de tip ATOM) sau fișierele publicate pe data.gov.ro.

## Declarație privind caracterul oficial al datelor

Doar versiunea disponibilă prin setul de date publicat pe www.data.gov.ro, inclusiv prin serviciile oficiale asociate (de exemplu WMS și servicii INSPIRE de descărcare de tip ATOM), precum și prin website-ul oficial al Ministerului Mediului, Apelor și Pădurilor (MMAP), trebuie considerată ca fiind versiunea oficială.
